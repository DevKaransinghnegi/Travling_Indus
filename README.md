# 🌍 Static Traveling Website

A clean, modern, and fully responsive traveling website built to showcase core frontend development skills. This project highlights smooth navigation, structured layouts, and interactive UI elements.

## 🛠️ Features & Skills Demonstrated

* **Semantic HTML5:** Structured, accessible, and SEO-friendly web layout.
* **Modern CSS3:** Custom styling, Flexbox/Grid layouts, animations, and full mobile responsiveness.
* **Interactive JavaScript:** Dynamic UI elements (like mobile navigation menus, image sliders, or scroll effects).
